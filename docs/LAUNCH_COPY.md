## GitHub release description

ContextGO is a local-first context runtime for multi-agent AI coding teams.

What it does:

- Unifies Codex, Claude, shell history, and memory files into one searchable context chain
- Exposes one CLI for `search / semantic / save / serve / health / smoke / native-scan`
- Ships with health, smoke, benchmark, and installed-runtime validation
- Upgrades hot paths with Rust/Go without changing the operator interface

Suitable for teams that need local data boundaries, reproducibility, rollback, and lower token cost.
